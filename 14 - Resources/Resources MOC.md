## Learning Path

### Phase 1: Foundations (Months 1-3)
1. [[What is Algorithmic Trading]] → [[Financial Markets Overview]] → [[Asset Classes]]
2. [[Probability and Statistics for Trading]] → [[Time Series Analysis]]
3. [[Market Microstructure MOC]] → [[Order Book Dynamics]] → [[Liquidity]]
4. Python fundamentals → pandas, numpy, matplotlib

### Phase 2: Strategy Development (Months 3-6)
1. [[Momentum Strategies]] → [[Mean Reversion Strategies]] → [[Statistical Arbitrage]]
2. [[Backtesting Framework Design]] → [[Overfitting]] → [[Walk-Forward Analysis]]
3. [[Risk Management MOC]] → [[Position Sizing]] → [[Value at Risk (VaR)]]
4. Build first backtesting engine

### Phase 3: Advanced Topics (Months 6-12)
1. [[Stochastic Calculus]] → [[Options Strategies for Algos]] → Derivatives pricing
2. [[Machine Learning Strategies]] → [[Deep Learning]] → [[Reinforcement Learning]]
3. [[Modern Portfolio Theory]] → [[Factor Models]] → [[Black-Litterman Model]]
4. [[Market Making Strategies]] → [[High-Frequency Trading]]

### Phase 4: Production (Months 12+)
1. [[Infrastructure MOC]] → [[Execution MOC]] → Live trading systems
2. [[Regulation and Compliance MOC]] → [[Risk Management MOC]]
3. [[Transaction Cost Analysis]] → [[Smart Order Routing]]
4. Deploy live strategies with real capital
